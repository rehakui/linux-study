# 2026-06-06 KVMの基礎

## 学習内容

### KVMとは

KVM（Kernel-based Virtual Machine）は、Linuxカーネルに標準で組み込まれている仮想化システム。

- Kernel-based Virtual Machine の略
- Linuxカーネルの機能として提供される
- ハイパーバイザ型の仮想化技術
- CPUが仮想化支援機能（Intel VT-x、AMD-Vなど）に対応している必要がある

### libvirt

- KVMなどの仮想化環境を管理するためのソフトウェア
- 仮想マシンの作成、起動、停止などを統一的に管理できる
