# ファイルの圧縮と解凍

2026-07-16

## 学習内容

- Udemy Section3.28「ファイルの圧縮と解凍」

## 問われた内容・学んだこと

### gzipでファイルを圧縮する

```bash
gzip test.txt
```

- `test.txt`を圧縮し、`test.txt.gz`を作成する。
- 元のファイルは削除される。

### 元のファイルを残して圧縮する

```bash
gzip -k test.txt
```

- `-k`オプションを付けると、元のファイルを残したまま圧縮できる。

### gzipファイルを展開する

```bash
gzip -d test.txt.gz
```

- `.gz`ファイルを展開する。
- 展開後は通常、`.gz`ファイルは削除される。

## 実践

- `test.txt`を作成し、`gzip -k`で圧縮を実施。
- `ls`で`test.txt`と`test.txt.gz`の両方が存在することを確認した。
- `cat test.txt`で内容を確認した。
- `cat test.txt.gz`では圧縮ファイルのため内容を読むことができないことを確認した。
- `gzip -d test.txt.gz`で解凍し、`test.txt`のみが残ることを確認した。

## 気づき

- `gzip`は圧縮時・解凍時ともに、デフォルトでは元ファイルを削除する。
- 元ファイルを残したい場合は`-k`オプションを使用する。
- `.gz`ファイルはテキストファイルではないため、`cat`では内容を正常に表示できない。
