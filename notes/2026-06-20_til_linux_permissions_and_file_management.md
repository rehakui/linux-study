## ファイルとディレクトリの所有者

Linuxのファイルやディレクトリには以下が設定されている。

* 所有者（Owner）
* 所有グループ（Group）

アクセス権は所有者・グループ・その他ユーザーごとに設定できる。

---

## アクセス権

### 3つのアクセス権

* 読み取り（r）
* 書き込み（w）
* 実行（x）

### アクセス権の表記

```text
-rwxr-xr--
```

例：

* rwx : 所有者
* r-x : グループ
* r-- : その他ユーザー

---

## 特殊なアクセス権

### SUID（Set User ID）

実行時にファイル所有者の権限でプログラムを実行する。

### SGID（Set Group ID）

実行時にファイル所有グループの権限でプログラムを実行する。

## ファイル管理コマンド

| コマンド  | 概要                  |
| ----- | ------------------- |
| ls    | ファイルやディレクトリの一覧を表示   |
| file  | ファイルの種類を確認          |
| cp    | ファイルやディレクトリをコピー     |
| mv    | ファイルやディレクトリを移動・名前変更 |
| mkdir | ディレクトリを作成           |
| rm    | ファイルやディレクトリを削除      |
| touch | 空ファイルの作成、タイムスタンプ更新  |
