## 学習内容
- どのようなOS
    tux(タックス) 微妙に顔が違うことがあるらしい（笑）
    リーナストーバルズ
    unixを目指して開発　unix系の一つ、macosも　兄弟子と弟弟子みたいな

    Linuxはosのカーネル部分を指す
    つまり、リーナストーバルズはカーネルを作った

- メリットとデメリット
    メリットとして、フリーでオープンソース
    そのためサーバや家電などに使われる

- ディストリビューションの意味と種類
    red hat 系(RHEL)
    Debian 系(Ubuntu)

- GUIとCUI
    CUI(Character-based User Interface)
    character 文字という意味
    GUI(Graphical User Interface)
