# TIL 2026-06-24 Day2（パスとリンク）

## パス

### 絶対パス

`/`（ルートディレクトリ）から始まるパス。

例

```bash
/home/penguin/file.txt
```

### 相対パス

現在のディレクトリを基準にしたパス。

よく使う記号

* `.` : 現在のディレクトリ
* `..` : 1つ上のディレクトリ
* `~` : ホームディレクトリ

---

## cdコマンド

```bash
cd
```

または

```bash
cd ~
```

はホームディレクトリへ移動する。

例

```bash
/home/penguin
```

`/home` はユーザーのホームディレクトリが格納されている場所であり、ホームディレクトリそのものではない。

---

## ハードリンク

作成

```bash
ln file1 hard1
```

特徴

* 元ファイルと同じiノードを共有する
* 同じ実体に別名を付けるイメージ
* 元ファイルを削除しても利用できる

---

## シンボリックリンク

作成

```bash
ln -s file1 sym1
```

特徴

* ファイルへのパスを保持する
* ショートカットのようなもの
* 元ファイルを削除するとリンク切れになる

---

## まとめ

* 絶対パスは `/` から始まる
* 相対パスは現在地を基準にする
* `cd` と `cd ~` はホームディレクトリへ移動する
* ハードリンクは同じiノードを共有する
* シンボリックリンクはパスを保持するショートカット
* ハードリンク = 同じ実体に別名を付ける
* シンボリックリンク = 実体への道順を記録する
