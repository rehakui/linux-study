# TIL - Linuxの起動とシャットダウン

## Linuxの起動の流れ

1. 電源を入れる
2. BIOS / UEFI が起動する
   - BIOS / UEFI はマザーボード上にあるファームウェア
   - ブートローダーを実行してOSを起動する
3. Linux（OS）が起動する
4. ユーザーを選択し、パスワードを入力してログインする
   - GUI（画面操作）
   - CUI（コマンド操作）

## シャットダウンについて

Linuxは複数人で利用することを前提としているため、誰でも自由にシャットダウンできるわけではない。

シャットダウンには管理者権限（root権限）が必要。

### Debian系Linuxのシャットダウン

```bash
sudo shutdown -h now
